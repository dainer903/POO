Descripción breve de cada ejercicio:

Ejercicio 1: Sistema de Usuarios

Este ejercicio consiste en crear un sistema que gestione diferentes tipos de usuarios dentro de una institución.
Se diseñó una clase base Usuario, de la cual heredan las clases Administrador, Profesor y Estudiante.
Cada tipo de usuario tiene atributos y métodos propios, aplicando los principios de herencia, constructores y sobrescritura de métodos.

Ejercicio 2: Zoológico

En este ejercicio se modeló un sistema para registrar animales de un zoológico.
Se creó una clase base Animal, que se divide en dos subclases principales: Mamifero y Reptil.
A partir de estas se desarrollaron clases específicas como León, Gato, Perro, Serpiente y Cocodrilo.
El proyecto demuestra el uso de herencia multinivel, polimorfismo y encapsulamiento, además de mostrar cómo los métodos sobrescritos permiten comportamientos distintos según la especie.
